# Phase2 UI/Login Mode Patch

변경 내용:

1. `Supabase 연결됨` 문구를 사용자 UI에서는 `서버 연결됨`으로 변경했습니다.
2. 로그인 상태에서 왼쪽 JS 플레이어와 오른쪽 차트 영역의 높이를 맞추고, 왼쪽 플레이어 패널 자체의 외부 스크롤을 제거했습니다.
3. 비로그인 상태에서는 왼쪽 풀 플레이어/Cloud Playlist 패널을 숨깁니다.
4. 비로그인 상태에서도 차트 앨범 이미지를 클릭하면 단일 곡 재생/일시정지만 가능합니다.
5. 비로그인 상태에서는 선택/플레이리스트 추가 컬럼을 숨깁니다.

적용 방법:

- ZIP 압축을 풀어 기존 GitHub 저장소에 덮어씌웁니다.
- commit / push 후 Streamlit 앱을 Reboot합니다.

주의:

- 로그인 후에는 기존처럼 왼쪽 풀 플레이어와 Cloud Playlist 저장/불러오기/삭제가 표시됩니다.
- 비로그인 모드는 앱 내부 localStorage playlist 저장을 하지 않도록 막았습니다.
